# Vrinda Annual Sales Analysis

## Overview
This project analyzes Vrinda Store’s annual sales data to identify customer behavior, top-performing regions, and sales channels. The goal is to derive actionable insights to improve overall sales.

## Data Processing
- Removed missing and duplicate values  
- Standardized gender values (Men/Women)  
- Created Age Group column (Teenager, Adult, Senior)  
- Converted amount column to numeric format  
- Checked inconsistent state names  
- Created calculated columns for analysis  
- Cleaned and validated dataset  

## Key Insights
- Women are more likely to purchase compared to men (~65%)
- Maharashtra, Karnataka, and Uttar Pradesh are the top 3 states (~35%)
- Adult age group (30–49 years) contributes the most (~50%)
- Amazon, Flipkart, and Myntra channels are max contributing (~80%)

## Final Conclusion
Target **women customers aged 30–49 years** living in **Maharashtra, Karnataka, and Uttar Pradesh** by showing **ads, offers, and coupons** on **Amazon, Flipkart, and Myntra**.
